# Cesta Mais Frontend

Este é o repositório do frontend do projeto **Cesta Mais**, desenvolvido com Vue 3, Vite e TypeScript.

## 🚀 Tecnologias Utilizadas

- **Vue 3**: Framework progressivo para construção de interfaces de usuário.
- **Vite**: Build tool extremamente rápida.
- **TypeScript**: Superset de JavaScript que adiciona tipagem estática.
- **Pinia**: Gerenciamento de estado intuitivo e tipado para Vue.
- **Vue Router**: Roteamento oficial para Vue.js.
- **Axios**: Cliente HTTP baseado em promessas para o navegador e node.js.

## 🛠️ Instalação e Execução

Para rodar o projeto localmente, siga os passos abaixo:

1.  **Clone o repositório:**
    ```bash
    git clone https://github.com/nilopadilha/appsolivoscestamais-web.git
    cd appsolivoscestamais-web
    ```

2.  **Instale as dependências:**
    ```bash
    npm install
    ```

3.  **Inicie o servidor de desenvolvimento:**
    ```bash
    npm run dev
    ```

4.  **Para gerar a build de produção:**
    ```bash
    npm run build
    ```

## 📂 Estrutura do Projeto

- `src/assets`: Contém imagens e outros arquivos estáticos.
- `src/components`: Componentes Vue reutilizáveis.
- `src/App.vue`: Componente raiz da aplicação.
- `src/main.ts`: Ponto de entrada da aplicação.
- `public/`: Arquivos estáticos servidos diretamente.
